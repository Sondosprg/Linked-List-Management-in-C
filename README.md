## Description

This project is a comprehensive educational application in **C** language designed for managing linked lists using a variety of fundamental and advanced functions.

>  **Personal Note:**
> This project is **entirely my own design** and was developed **completely individually**. A significant portion of the work was initially done using **only a mobile phone** via a C coding application, before I acquired a desktop computer.

---

## Features

### List Creation
- `creationfifo()`: Creates a list using the FIFO (First-In, First-Out) system.
- `creationlifo()`: Creates a list using the LIFO (Last-In, First-Out) system.

### Display
- `display()`: Displays all elements.

### ➕ Add Elements
- `addbeing()`: Adds to the beginning of the list.
- `addend()`: Adds to the end of the list.
- `addpos()`: Adds at a specific position.

### ➖ Delete Elements
- `deletebegin()`: Deletes the first element.
- `deleteend()`: Deletes the last element.
- `deletepos()`: Deletes from a specific position.
- `deletelist()`: Deletes the entire list.

### Additional Operations
- `link()`: Links two lists.
- `tabtotlistposneg6()`: Converts an array into two lists based on a condition.
- `tabtosortlist6()`: Sorts list elements via a temporary array.
- `tabtolist()`: Converts an array to a list.
- `listtotab()`: Converts a list to an array.
- `sortasc()`: Sorts the list in ascending order.

## الميزات

### إنشاء القائمة
- `creationfifo()`: إنشاء قائمة بنظام FIFO (أول ما يدخل أول ما يخرج).
- `creationlifo()`: إنشاء قائمة بنظام LIFO (آخر ما يدخل أول ما يخرج).

### العرض
- `display()`: عرض كافة عناصر القائمة.

### ➕ إضافة عناصر
- `addbeing()`: إضافة عنصر في بداية القائمة.
- `addend()`: إضافة عنصر في نهاية القائمة.
- `addpos()`: إضافة عنصر في موقع معين.

### ➖ حذف عناصر
- `deletebegin()`: حذف العنصر الأول.
- `deleteend()`: حذف العنصر الأخير.
- `deletepos()`: حذف عنصر من موقع معين.
- `deletelist()`: حذف كامل القائمة.

### عمليات إضافية
- `link()`: ربط قائمتين متصلتين.
- `tabtotlistposneg6()`: تحويل مصفوفة إلى قائمتين بناءً على شرط معين.
- `tabtosortlist6()`: فرز عناصر القائمة باستخدام مصفوفة مؤقتة.
- `tabtolist()`: تحويل مصفوفة إلى قائمة متصلة.
- `listtotab()`: تحويل قائمة متصلة إلى مصفوفة.
- `sortasc()`: فرز القائمة تصاعديًا.
